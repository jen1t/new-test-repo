# javascriptproject

A tiny JavaScript demo project that prints "Hello, World!" and a few sample expressions, and includes an interactive age calculator.

Files
- hello-world.js — example that logs messages, arithmetic, and prompts for birth year to compute age.

Running

1. Install Node.js if not installed: https://nodejs.org/
2. Run:

   node hello-world.js

   The program will prompt for your birth year (e.g., 1990) and then print your age.
